2013年2月5日
Version 0.0.5_20130205
1.初步完成情景模式的基本功能
1.1 显示4个默认的预设模式：静音，会议，标准，户外
1.2 可添加自定义的情景模式
1.3 可重置当前情景模式到初始状态
1.4 点击每个情景模式进入详细项目编辑界面
1.5 可删除非预设模式和非激活状态的情景模式
1.6 可通过单击情景模式列表实现激活情景模式
2.初步完成最基本的选项设置
2.1 可修改情景模式名称
2.2 可设置音量(多媒体音量，来电铃声音量，通知铃声音量，闹钟铃声音量)
2.3 可设置来电是否振动
2.4 可设置来电铃声和通知铃声
